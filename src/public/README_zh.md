有一个关于白宫会议的引人入胜的故事，当时的瑞典首相斯特凡·勒文和乔·拜登会面。当他们坐在顾问、战略家和专家的包围中时，拜登以熟悉的缓慢语调向勒文倾身说道："你知道，斯特凡，我们可能是这个房间里唯一真正知道来自工人阶级是什么感觉的人。"

截至2024年8月，美国的工资增长一直超过通货膨胀。具体而言，
工资同比增长4.6%，而同期通胀率为2.5%。
