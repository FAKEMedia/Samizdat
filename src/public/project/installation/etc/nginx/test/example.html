English

This is the default (English) version of the example page. You're seeing this content because:

  1. Either your browser has no language cookie set, or it's set to en
  2. Nginx is configured with i18n support
  3. The filename matches the default pattern example.html

How It Works

Nginx checks the language cookie in your request and tries to find a file with the corresponding language suffix (like _sv for Swedish) before falling back to the default version.

You can switch languages by changing the cookie in your browser. For example, in Developer Tools > Console: 

document.cookie = "language=sv; path=/; max-age=31536000"
