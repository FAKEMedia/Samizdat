# Samizdat

Samizdat es parte de un proyecto para crear HTML estático a partir de archivos markdown subidos a GitHub.
Con el tiempo, se han añadido más funciones de alojamiento web.
Un subobjetivo es probar la contabilidad abierta, donde las cifras pueden presentarse continuamente después de ser anonimizadas.

Esta integración maneja la facturación de forma independiente y sincroniza las facturas con Fortnox.
Se almacenan datos mínimos del cliente local en Fortnox.
Los pagos registrados y las facturas de proveedores se recuperan de Fortnox y se muestran en la interfaz del cliente local.
Se puede configurar para mostrar todo públicamente también.
